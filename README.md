## 数据结构
红黑树是一种含有红黑节点，并能自平衡的二叉查找树。必须满足以下性质：
1. 每个节点非黑即红。
2. 根节点是黑色
3. 每个叶子节点（NIL）是黑色
4. 每个红色节点的两个子节点一定都是黑色
5. 任意一节点到每个叶子节点的路径都包含数量相同的黑节点。
